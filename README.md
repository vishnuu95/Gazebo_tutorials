# Gazebo_tutorials
Week 12 assignment. ENPM 808X. Software Development for Robotics. 

## Running
```
cd catkin_ws/src
git clone https://github.com/vishnuu95/Gazebo_tutorials
cd ..
catkin_make
source devel/setup.bash
```
For launching
```
roslaunch Gazebo_tutorials turtlebot.launch recordFlag:=False
```
